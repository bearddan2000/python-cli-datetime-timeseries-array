# python-cli-datetime-timeseries-array

## Description
A demo of creating datetime series using start date, number of days, and filtering for only M-F.

## Tech stack
- datetime
- dateutil

## Docker stack
- python:latest

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## To see help
`sudo ./install.sh -h`
